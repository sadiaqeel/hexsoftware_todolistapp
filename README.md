# 📝 Interactive To-Do List | Hex Softwares Project

A modern, functional To-Do List application built during my tenure at **Hex Softwares**. This project demonstrates core front-end development principles, including state management, persistent browser storage, and dynamic UI rendering.

## 🚀 Key Features
- **Task Management:** Add, complete, and delete tasks seamlessly.
- **Data Persistence:** Integrated `localStorage` API so your tasks are saved even after closing the browser.
- **Smart Filtering:** View tasks by status: *All*, *Pending*, or *Completed*.
- **Bulk Actions:** "Clear All" functionality with a built-in user confirmation dialog.
- **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack
- **HTML5:** Structured for accessibility and SEO.
- **CSS3:** Custom styling featuring a clean, minimalist UI and smooth transitions.
- **JavaScript (ES6+):** - DOM Manipulation
  - Event Handling
  - Array Filtering & Logic
